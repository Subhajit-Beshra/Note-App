# 📝 Notes App

A simple web-based Notes Application built with **HTML, CSS, and JavaScript** that allows users to **add, view, and delete notes**. Notes are stored locally in the browser using **localStorage**, making it fast, lightweight, and easy to use.

---

## Features

- Add new notes with a title or text.
- View all saved notes in a clean, organized layout.
- Delete individual notes with a single click.
- Notes are **persisted in localStorage**, so they remain after page reload.
- Lightweight and fast — no backend required.
- Fully responsive design for desktop and mobile.
